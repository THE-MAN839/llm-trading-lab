How It Works

1. Live Paper Trading

Production strategy runs on Alpaca's paper trading API:

  1. scheduler.py runs every 30 minutes during market hours
  2. Calls live_trade.py to execute strategy
  3. Strategy reads from src/strategy.py
  4. Checks RSI signals, places trades via Alpaca API
  5. Logs all trades to logs/live_trades.jsonl

2. Research Agents

Each agent runs experiments in isolation:

  1. Agent reads their strategy.py
  2. Agent hypothesizes improvement
  3. Agent modifies parameters
  4. Agent runs python3 research.py
  5. Results logged to logs/research_history.jsonl
  6. Repeat

3. Production Updates

When agents find improvements:

  1. Human reviews agent findings
  2. Best config copied to src/strategy.py
  3. Live trading picks up changes on next cycle
